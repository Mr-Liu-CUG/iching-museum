## ✦ 项目概览

这是一座建在浏览器中的**数字易经博物馆**。它不止于展示六十四卦的符号与卦辞，而是一次完整的易学文化导览——

- **序厅（Prologue）**：从伏羲画卦到宋明理学的时间轴，从大衍筮法到金钱卦的操作指南，从王弼到傅佩荣的四位大师，从莱布尼茨到荣格的古今对话
- **核心展厅**：64 卦全景矩阵、六爻交互变卦、卦象关系图谱、诸家解卦对比、现代五维生活解读
- **四语支持**：简体中文、繁體中文、English、日本語 自由切换

**零依赖，零构建。双击 `index.html` 即可运行。**

            ┌────────────────┼────────────────┐
            │                │                │
            ▼                ▼                ▼
   ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
   │ hexagrams-   │ │ data/lang/   │ │ data/content-│
   │ data.js      │ │ {lang}.js    │ │ {lang}.js    │
   │ (12,625 行)  │ │ (4 文件)     │ │ (3 文件)     │
   │              │ │              │ │              │
   │ 64 卦完整数据 │ │ UI 标签翻译   │ │ 内容翻译     │

### 双轨 i18n 架构

项目采用 UI 层与内容层分离的双轨国际化方案：

```
I18n.t(key)           → UI 标签（导航、按钮、卡片标题）
                        通过 data-i18n 属性注入 DOM
                        切换语言时自动刷新所有元素

I18n.content(hex, path) → 内容翻译（卦名、卦辞、爻辞、现代解释）
                        通过 CONTENT_XX 查找表
                        缺失时自动回退中文源数据
```

**设计亮点**：

| 特性 | 实现 |
|------|------|
| 零开销源语言 | zh-CN 直接从 `HEXAGRAMS_DATA` 读取，无翻译查找 |
| 繁体运行时转换 | zh-TW 使用 S→T 字符映射表（~350 对），无需存储完整繁体副本 |
| 按需加载 | content-en.js / content-ja.js 由 `<script>` 标签动态注入 |
| 优雅降级 | 翻译缺失字段自动回退中文源数据 |
| `file://` 兼容 | 全程 `<script>` 标签加载，无需 HTTP 服务器 |

### 数据索引

数据结构使用**伏羲先天二进制序**作为主索引：

```
六位二进制 → 十进制 (0-63)
000000 = 坤为地  (index 0)
000001 = 山地剥  (index 1)
...
111110 = 泽天夬  (index 62)
111111 = 乾为天  (index 63)
```

`kingWen` 字段维护通行卦序（1-64），slider 通过 `kwLookup()` 进行 O(1) 映射。

### 字体

- **标题/正文**：Noto Serif SC（宋体风格衬线）— 仿宋刻本
- **辅助文字**：Noto Sans SC — 现代易读
- **日文**：Noto Serif JP — 和刻本风格
- **等宽备用**：KaiTi / SimSun / STSong

### 排版

- 行高 `1.7`–`2.0` — 仿古籍疏朗排布
- 字间距 `0.05em` — 宋版书气韵
- `text-align: justify` — 两端对齐
- 金色边框 `1.5px solid rgba(176, 141, 87, 0.25)` — 宋代描金工艺

### 动画

| 缓动曲线 | 用途 |
|----------|------|
| `cubic-bezier(0.16, 1, 0.3, 1)` | 卡片入场、面板展开 |
| `cubic-bezier(0.34, 1.56, 0.64, 1)` | 按钮悬停回弹 |
| 爻线涟漪 + `scaleX` 脉冲 | 模拟点击竹简触感 |

### 装饰元素

- `❖` 菱形节点 — Timeline 标记、卡片图标
- `☯` 阴阳符号 — 分隔、印章
- `⚊ ⚋ ☰ ☱ ☲ ☳ ☴ ☵ ☶ ☷` 八卦符号 — 卦象显示
- `䷀` ䷁ 等 — 六十四卦 Unicode 字符

## ✦ 无障碍设计

项目遵循 WAI-ARIA 1.1 规范：

- **Tab 切换**：`role="tablist"` → `role="tab"` + `aria-selected` + `aria-controls` ↔ `role="tabpanel"` + `aria-labelledby`（序厅和诸家解卦两套 Tab 均实现双向语义关联）
- **展开按钮**：`aria-expanded` 动态切换于序厅通识展板
- **动态内容区**：`aria-live="polite"` — 屏幕阅读器实时播报关卡更新
- **语言标识**：`html[lang]` 属性随语言切换自动更新
- **装饰元素**：所有纯装饰性符号标记 `aria-hidden="true"`

## ✦ 使用方式

### 快速启动

```bash
# 方式一：直接打开（推荐）
# 双击 index.html 即可在浏览器中运行

# 方式二：本地服务器（可选）
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 语言切换

点击页面右上角的 `简 | 繁 | EN | 日` 即可切换。

- **简体中文**：完整内容，从源数据直接渲染
- **繁體中文**：运行时 S→T 字符转换
- **English**：完整英语翻译（卦名、卦辞、爻辞、现代解释、诸家基础字段）
- **日本語**：卦名 + 现代五维解释为日文，其余字段优雅回退中文

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `←` | 切换到前一卦（伏羲先天序递减） |
| `→` | 切换到后一卦（伏羲先天序递增） |
